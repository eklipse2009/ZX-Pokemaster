
This is the ultimate, most complete ZX Spectrum files set that have ever been!

This set contains:
 - All files that were in TOSEC;
 - All files from www.worldofspectrum.org which were never in TOSEC before;
 - All modern homebrews up to September 2017;
 - Demos from pouet.net;
 - Games which took part in Crap Games Competition;
 - Rare hacks and compilations.
 
Provided by Lady Eklipse with love to all ZX Spectrum community.

31. October, 2017

If you want an easy way to sort these files any way you want, please check out ZX Pokemaster:
https://sourceforge.net/projects/zx-pokemaster/

Представляю Вашему вниманию самую крупную раздачу игр и программ для ZX Spectrum за всю историю существования сцены!

Эта раздача - результат моего многомесячного труда по сортировке и переименованию файлов ZX Spectrum для проекта TOSEC.

Она включает в себя:
 - Все файлы, которые были в TOSEC-сетах до этого момента;
 - Все файлы с файлообменника worldofspectrum.org, которые никогда ранее не были в TOSEC;
 - Все современные игры homebrew вплоть до сентября 2017 года;
 - Все демки с pouet.net;
 - Все игры, участвовавшие в Crap Games Competition с 1996 по 2017 год;
 - Все дампы кассет с spectrum4ever.org;
 - Коллекции игр и демок с zxaaa.net;
 - И не только.
 
С любовью к комьюнити ZX Spectrum,
Эля aka Lady Eklipse.

Я являюсь разработчицей программы ZX Pokemaster, созданной специально для сортировки и переименования файлов для ZX Spectrum, но в первую очередь - для автоматизации создания файлов с чит-кодами - .POK. Очень рекомендую воспользоваться после скачивания раздачи, чтобы рассортировать файлы так, как Вам по душе.
